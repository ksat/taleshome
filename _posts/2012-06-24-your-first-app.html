--- layout: firstapp title: Install Tales secondtitle: firstapp permalink: /your-first-app --- Make your first quick tales app
In this example we will build a simple Hello world page:

Before you begin, make sure you have tales installed.
Step 1: Create the app and run
Open terminal. Go to the directory where you want to create the app. And type 
>fan tales new hello
You should now see a hello folder created in the current directory
cd to that directory using 
>cd hello
Now run the server using
>fan tales run
Open your browser and point to http://localhost:8000.You should see the "Welcome to Tales" message. Its that easy to get started
Step 2: Print Hello world
Open fan/Index.fan and edit it as follows:

fan/Index.fan 
using tales
class Index : Page{
  @Route{uri = "/"}
  Void main(){
    response.writeStr("Hello world")
  }
}
Now refresh your page in browser. You should see a hello world message
Step 3: Print Hello world using a Template
Open template/Index.html, remove everything and change code to the following:

template/Index.html 
Hello world, time is: <span talesId="time"></span>
The code is self explanatory. Just note how talesId is defined in one of the tags. It defines a "slot" for dynamic data to be populated
Now Open Index.fan, remove everything and change code to the following:

fan/Index.fan 
using tales
class Index : Page{
  @Route{uri = "/"}
  Void main(){
      html := Html("template/Index.html")
      html.tag("time").text(DateTime.now.toLocale("hh:mm:ss aa"))
      response.writeTag(html)
  }
}
This time instead of writing a string we write a Tag. Note:
  1. The html tag is create with a file, pointing to Index.html
  2. In the slot (with talesid) "time", we add text with current date
