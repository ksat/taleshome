Https
SSL support in Tales is as simple as adding "cert" and "key" files to cert/ directory
To get started, create a cert/ directory directly under your app directory and copy the "key" and "cert" files into it.
You can generate the cert files using openssl like this 
openssl genrsa 1024 > host.key
openssl req -new -x509 -nodes -sha1 -days 365 -key host.key > host.cert
Now open fan/Settings.fan and add a Https Setting like this 
using tales

class Settings : tales::Settings{
    override Void main(){
      add(HttpsSetting{})
    }
}
You can pass additional parameters to HttpsSetting 
add(HttpsSetting{
  port = <port>
  keyFile = <key-file-location>        
  certFile = <cert-file-location>
  certPassword = <certificate-password>
})
