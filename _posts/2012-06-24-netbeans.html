Setting up netbeans IDE
Create a new tales project using "fan tales new" command 
fan tales new newapp
An app directory newapp/ is created for you
  1. Select File->Open Project
  2. Select your project folder "newapp/"
  3. Choose menu Run->Build Project
  4. Choose menu Debug->Debug Project
That's it. now Just modify the classes as usual and debug.
