Custom Error and NotFound Pages
Custom error and notfound pages can be displayed using the CustomPageSetting
To define custom error and notfound pages open Settings.fan and add a CustomPageSetting like this 
add(CustomPageSetting{
  notFound = Index#.method("notFound")
  error = Index#.method("error")
})
Now, When a page is not found the request will be routed to "notFound" method in the class Index and when an error occurs the request will be routed using the "error" method in class Index.
